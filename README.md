## ✨ About the App

CozyDo helps you stay productive while feeling calm and organized.  
It combines **beautiful UI + smooth animations + helpful productivity tools**.

---

## 📱 Features

- 📝 Add, edit, delete tasks  
- ☑️ Mark tasks as completed  
- 📂 Categories (Work, Study, Personal)  
- 🔍 Smart search & filters  
- 🌙 Dark mode support  
- 📳 Haptic feedback  
- 🎉 Confetti animation on completion  
- 🔊 Sound effects for actions  
- ⚡ Smooth transitions & animations
